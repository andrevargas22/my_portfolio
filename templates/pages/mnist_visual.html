The pipeline behind it

The model is trained on MNIST, versioned in the MLflow Model Registry, and deployed to Google Cloud Run as a containerised FastAPI service. Your drawing flows from the canvas to that production endpoint, which returns predictions and internal activations for the live 3D view.
